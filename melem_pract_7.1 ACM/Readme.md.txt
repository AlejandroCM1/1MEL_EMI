El proyecto consiste en una calculadora simple que usa como procesador una placa de Arduino Nano. 
Los números a operar, así como las operaciones se introducen a través de un pequeño teclado.
En la pantalla LCD se registra cada nuevo dato, así como el resultado final de la operación.

La calculadora puede operar con dos números de dos cifras, soportando las operaciones de suma, resta, 
multiplicación y división.

Dispone también de un botón de reset, que tras pulsarlo, permite realizar otra operación.
